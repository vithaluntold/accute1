AI Overview
Zoho Forms provides a wide variety of UI form components, categorized into basic fields, advanced fields, and tools for organizing your layout. 
Basic fields
Single line: A text field for concise, single-line input.
Multi-line: A larger text area for more extensive, paragraph-style input.
Number: Collects numerical data only.
Decimal: For collecting numerical data that can include decimal points.
Name: A composite field with sub-fields for title, first name, middle name, and last name.
Address: A structured, composite field for collecting comprehensive address information with sub-fields for street, city, state, and zip code.
Phone: Collects phone numbers, with options for international formatting and country codes.
Email: Gathers email addresses and can be configured with domain validation.
Date: A calendar-based field for selecting and inputting a date.
Time: Used to collect time values and can be set to 12-hour or 24-hour formats.
Date-time: Combines date and time selection into one field.
Month-Year: Restricts the date selection to only the month and year.
Dropdown: A list of choices from which the user can select a single option. It can also be configured with searchable options.
Radio: Displays a set of options as radio buttons for a user to select one choice.
Multiple choice: A choice field that allows the user to select one or more answers from a list of predefined options.
Checkbox: A choice field similar to Multiple Choice, also allowing users to select multiple options.
Decision box: A simple checkbox field for a "yes" or "no" type response.
Website: Validates input to ensure it is a proper URL.
Currency: Gathers monetary values and allows for different currency units.
Slider: A visual component that lets users select a value from a predefined range.
Rating: Allows users to provide feedback using a star rating scale. 
Advanced fields
Image choice: A set of choices that are represented by images.
Matrix choice: An advanced choice field that lets you group choices and choice types under different sections.
File upload: Allows users to upload and attach files to the form submission.
Image upload: Specifically for uploading image files.
Audio/Video upload: Allows users to either upload audio/video files or record them directly in the form.
Signature: Captures a digital signature from the user.
Subform: Enables a one-to-many relationship between data, allowing a form to be embedded within another.
Unique ID: Automatically generates a unique, sequential number for every form entry.
Random ID: Generates a random, unique ID for every form entry.
Formula: Automatically computes and generates a value based on a predefined formula.
Lookup: Fetches and displays data from another form.
Zoho CRM field: Allows you to access and prefill data from a Zoho CRM module.
Image embed: Lets you embed an image directly into the form.
Audio/Video embed: Allows you to embed an audio or video file directly into the form. 
Layout tools
Section: Divides the form into logical groups for better organization.
Description: A component for adding descriptive text or instructions to the form.
Page break: Separates a long form into multiple pages.
Terms and conditions: A field that can be used to add your organization's terms and conditions.
