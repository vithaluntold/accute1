**Q: How do you count users?**  
A: Any employee who creates or manages workflows = 1 user. Workflow execution doesn't require a user license.

**Q: What if we only need automation for one department?**  
A: Perfect for Professional tier. Start with 10-20 users in that department, expand as needed.

**Q: Can workflows call workflows?**  
A: Yes, unlimited nesting and complexity. No per-workflow fees.

**Q: What about API rate limits?**  
A: Professional: 1,000 API calls/hour. Enterprise: 10,000/hour. Enterprise Plus: Custom.

**Q: Do you charge for failed workflow executions?**  
A: No. Retries, failures, and tests are all unlimited.

**Q: Can we build custom AI agents?**

**Q: What integrations are included?**

**Q: What happens to our data if we cancel?**
